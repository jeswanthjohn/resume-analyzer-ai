# Resume Analyzer AI

An AI-powered web application that analyzes resumes using PDF parsing and OpenAI to provide ATS-style feedback.

## Features

- Upload resume PDF
- Extract resume text
- AI-powered analysis
- ATS score and improvement suggestions

## Tech Stack

- Node.js
- Express
- OpenAI API
- pdf-parse

## Setup

1. Clone the repo
2. Run `npm install`
3. Create `.env` using `.env.example`
4. Run `npm start`
